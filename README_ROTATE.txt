AutoTypeText.cs
-게임 내에서 메시지 박스에서 순차적으로 메시지를 출력하는 코드

BoxOverLapCheck.cs
-플레이어가 박스에 깔렸을때를 확인하고 게임을 다시 시작하는 코드

Clearzone.cs
-플레이어가 클리어 존에 도달했을때 다음 레벨을 시작하고 레벨 클리어 정보를 저장하는 코드

DeadZone.cs
-플레이어가 게임 플레이 존에서 나갔을때 게임을 다시 시작하는 코드

FadeIn.cs
-게임이 시작될때 화면의 밝기를 서서히 증가시키는 코드.

FadeOut.cs
-게임의 스테이트가 변경될때 화면의 밝기를 서서히 감소시키는 코드.

LevelNumGenerator.cs
-레벨 선택 화면에서 레벨에 맞는 번호를 생성하는 코드

OptionSceneActions.cs
-옵션 씬에서 사용되는 버튼 액션들

PauseActions.cs
-게임 일시 정지 씬에서 사용되는 버튼 액션들

PauseManage.cs
-게임 일시 정지 상태를 관리하는 코드

PlayerInputManage.cs
-플레이어가 월드를 회전시키고 있거나, 게임이 일시 정지중일때 플레이어를 제어하는 코드.

RotationInfomation.cs
-트리거의 정보를 받아서 월드를 회전 시키는 코드.

TriggerLogic.cs
-플레이어와 트리거사이의 충돌 처리를 하는 코드
